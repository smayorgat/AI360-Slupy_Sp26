Итоговая оценка складывается из:
- посещаемость (5 % и см. ниже),
- контрольная работа (25 %),
- работа в семинарах (20 %),
- конспектирование лекций и семинаров (20 %),
- заключительный проект (30 %).

Конспект занятий должен быть сдан преподавателю не позже чем через одну неделю после лекции.

**Оценка за курс**: если ваш итоговый балл равен _x_ (от 0 до 100), то:

* оценка A (отлично) при _x_ ≥ 87,
* оценка B (хорошо) при 87 > _x_ ≥ 73,
* оценка C (удовлетворительно) при 73 > _x_ ≥ 60,
* оценка D (неудовлетворительно) при 60 > _x_ ≥ -∞.

в другую дату, вам будет предоставлена новая КР, с другими задачами.

## 2. Посещаемость

Посещаемость составляет 5% от конечной оценки и **для допуска к заключительному экзамену необходимо посетить (с начала до конца)
хотя бы девять блоков занятий** (один блок занятие — это две лекционные пары и соответствующий семинар).

## 3. Семинары (практики), домашние задания и содержание КР

В каждом семинаре студенты будут решать на доске нерешенные задачи из предыдущего блока занятий. Результаты 
такой работы входят в строку "работа в семинарах" выше. Также во время семинаров (возможно иногда и во время
лекций) будут даваться или отмечаться задачи, которые войдут в список задач для контрольной работы.
